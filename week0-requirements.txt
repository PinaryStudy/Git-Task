# week0 과제 요구 사항입니다.

과제 실습과 같이 git config 설정을 하고 이를 확인합니다.

[ 요구 사항 ]
1. git user.name 이 설정되어야 한다.
2. git user.email 이 설정되어야 한다.

[ 과제 제출 방법 ]
위 요구 사항을 만족한 후 git 을 사용하는 터미널에서 다음 명령어를 사용합니다.
bash validate-week0.sh

해당 명령어를 사용한 뒤 출력된 내용을 모두 캡처하여 과제 제출 PR을 작성해주세요.
통과 하지 못한 부분이 있는 경우, 과제 제출로 처리하지 않습니다.
