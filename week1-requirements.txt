# week1 과제 요구 사항입니다.

실습 영상과 같이 진행해주세요.

[ 요구 사항 ]
1. <본인의 영어이름>.txt 파일을 생성한다.
2. git status 를 통해 상태를 확인한다.
3. git add 를 통해 변경사항을 스넵샷으로 저장한다.
4. git status 를 통해 나온 상태를 확인한다.
5. git commit을 통해 변경사항을 저장한다.
6. git status를 통해 나온 상태를 확인한다.
7. <본인의 영어이름>.txt 파일에 자유롭게 내용을 추가 한다.
8. git status를 통해 나온 상태를 확인한다.
9. git add 를 통해 변경사항을 스넵샷으로 저장한다.
10. git status를 통해 나온 상태를 확인한다.
11. git commit을 통해 변경사항을 저장한다.

[ 과제 제출 방법 ]
위 요구 사항을 만족한 후 git 을 사용하는 터미널에서 다음 명령어를 사용합니다.
bash validate-week1.sh

해당 명령어를 사용한 뒤 출력된 내용을 캡처하여 노션 과제 제출 방법을 참고하여 Issue 로 작성해주세요!

통과 하지 못한 부분이 있는 경우, 과제 제출로 처리하지 않습니다.
